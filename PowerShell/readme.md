**Install Modules**

Run Below Commands.

**Install-Module Microsoft.Xrm.Tooling.CrmConnector.PowerShell -Scope CurrentUser -Force -AllowClobber**

**Install-Module Microsoft.Xrm.Data.Powershell -Scope CurrentUser -Force -AllowClobber**
